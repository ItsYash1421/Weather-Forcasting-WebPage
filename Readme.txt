Project Title:Weather App

Description:
This project is a simple weather application that allows users to check the weather for a specific city and view the top 3 hottest cities. 
Additionally, it provides a toggle switch to switch between light and dark modes for better user experience.

Contents:
1. index.html - The HTML file containing the structure of the weather app and mode toggle.
2. styles.css - The CSS file containing styles for the weather app, mode toggle, and background image.
3. script.js - The JavaScript file containing the logic for fetching weather data,toggling between light and dark modes, and updating the UI accordingly.

Usage:
1. Open index.html in a web browser.
2. Enter the name of the city you want to check the weather for in the input field and click the "Search" button.
3. The weather information for the specified city will be displayed, along with the top 3 hottest cities.
4. Click the sun icon to switch to light mode, and click the moon icon to switch to dark mode.

Dependencies:
- This project requires an internet connection to fetch weather data from the OpenWeatherMap API.
- It also uses external icon images for the sun and moon toggle.

Credits:
- The weather data is fetched from the OpenWeatherMap API (https://openweathermap.org/).
- Sun and moon icons are sourced from icons8 (https://icons8.com/).
